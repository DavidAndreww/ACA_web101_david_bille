CSS: Trials and Tribulations
June 7th, 2019
The first week of class, we were already learning how CSS worked. Having barely touched on the key elements to begin an HTML document, we were already tackling a major language that has a very significant impact on what users see on the front end. Type, Attributes, Id, Class and Pseudo-class...the 5 selectors that make CSS do what it does. From an ameteaur's perspective, the values you could plug into these 5 selector types seemed endless. This post chronicles my first week trying to make sense of the countless pieces of code that I believed I had to know all at once.
