This second week of class has been a little bit more of a test. We have started to work on our own personal web pages including main landing page, resume and blog page. CSS and Git: The Crash Course
June 7th, 2019
The very first day of class, we were already learning how CSS worked. Having barely touched on the key elements to begin an HTML document, we were already tackling a major language that has a very significant impact on what users see on the front end. Type, Attributes, Id, Class and Pseudo-class...the 5 selectors that make CSS do what it does. We also learned about Git and GitHub, the tools that allowed for the projects from our local machines to be shared across the web. Not only share our projects and designs, but allow other developers to collaborate with us from anywhere in the world.
